# CAS-X Framework

Framework Name:
CAS-X (Causal Actionability Scoring Framework)

Purpose:
To identify SNP–gene relationships with sufficient causal evidence and experimental feasibility for CRISPR-based functional validation.

Scientific Problem:
GWAS studies identify disease-associated variants but do not reliably determine which SNP–gene relationships are sufficiently supported for functional CRISPR validation.

Unit of Analysis:
SNP → Gene relationship

Scoring Components:

1. GWAS Evidence (0–2)
- Replication
- Significance
- Fine-mapping support

2. Regulatory / Coding Evidence (0–3)
- Coding variant evidence
- Islet enhancer overlap
- Regulatory annotation

3. eQTL Evidence (0–3)
- Pancreatic islet eQTL
- Pancreas eQTL

4. Chromatin Evidence (0–3)
- Enhancer-promoter interaction
- Chromatin contact support

5. Expression Evidence (0–2)
- Pancreatic islet expression

6. Functional Evidence (0–3)
- Literature support
- Experimental validation

7. CRISPR Readiness (0–2)
- Perturbation feasibility
- Modality suitability

Maximum CAS-X Score = 18

Priority Tiers:

High Priority:
15–18

Moderate Priority:
10–14

Low Priority:
0–9

Output:
CAS-X Score
CRISPR Strategy
Priority Tier
