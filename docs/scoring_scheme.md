# CAS-X Scoring Scheme

## Layer 1: GWAS Evidence (0–3)
3 = Fine-mapped / replicated locus
2 = Genome-wide significant locus
1 = Suggestive evidence

## Layer 2: Regulatory Evidence (0–3)
3 = Strong enhancer overlap
2 = Moderate enhancer support
1 = Weak support

## Layer 3: Expression Evidence (0–2)
2 = High islet expression
1 = Moderate expression
0 = Low expression

## Layer 4: eQTL Evidence (0–3)
3 = Strong pancreas/islet eQTL
2 = Moderate eQTL
1 = Weak eQTL

## Layer 5: Chromatin Evidence (0–3)
3 = Direct chromatin interaction
2 = Probable interaction
1 = Weak interaction

## Layer 6: Functional Evidence (0–3)
3 = Published CRISPR/mechanistic evidence
2 = Functional evidence
1 = Literature support

CAS-X = GWAS + Regulatory + Expression + eQTL + Chromatin + Functional

Maximum Score = 17
