# CAS-X Project Plan

Title:
A Causal Evidence–Driven CRISPR Actionability Framework for Functional Prioritization of Type 2 Diabetes GWAS Loci

Framework:
CAS-X (Causal Actionability Scoring Framework)

Scientific Problem:
GWAS studies identify disease-associated variants but do not reliably determine which SNP–gene relationships are sufficiently supported for functional CRISPR validation.

Core Objective:
To integrate multiple causal evidence layers and prioritize SNP–gene relationships for CRISPR-based functional investigation.

Evidence Layers:
1. Genomic Architecture
2. Enhancer Evidence
3. eQTL Evidence
4. Chromatin Interaction
5. Islet Expression
6. Literature Support

Output:
CAS-X Score

Validation:
Known T2D genes should rank highly:
- TCF7L2
- SLC30A8
- KCNJ11
- PPARG
- CDKAL1
